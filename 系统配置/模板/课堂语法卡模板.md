# 课堂语法卡模板

用于创建或重建 `学习系统/语法` 下的语法卡。默认一个语法一张卡；只有紧密范式或敬语映射适合合并。

## 使用规则

- 创建前先搜索同名语法卡和真实存在的易混卡。
- `formation` 固定使用 YAML 列表，每种接续单独一项。
- 中文说明统一使用简体中文。
- 每个用法分支默认提供 2 条纯日语例句：一条基础句、一条自然场景句。复杂分支最多 3 条。
- 仅难句和对比例句附简短中文翻译。
- `contrast_with` 和正文 wikilink 只指向真实存在的卡。尚未创建但值得补充的对比卡写入 `## 待补卡`，使用普通文字。
- 核心含义或主要接续无法可靠确认时，暂停创建并向用户确认。`## 待确认` 只用于次要疑点。
- 用户直接要求新增且没有指定课堂笔记时，保持 `source_notes: []`，并在正文来源写 `用户直接录入；大模型整理`。
- 有课堂笔记来源时，将实际笔记 wikilink 写入 `source_notes` 和正文来源。

## 卡片模板

```md
---
track: class_review
item_type: grammar
status: active
priority: normal
done_today: false
pattern:
meaning_zh:
formation:
  -
source_notes: []
first_seen: YYYY-MM-DD
last_seen: YYYY-MM-DD
seen_count: 1
error_count: 0
review_stage: day0
next_review: YYYY-MM-DD
last_reviewed: ""
contrast_with: []
tags:
  - jp/grammar
  - jp/class_review
---
```

## 按需区块

有实际内容时，将以下区块放在 `## 来源` 之前；没有实际内容时省略。

```md
## 易错 / 易混

## 待补卡

- <尚不存在但值得后续创建的对比卡名称，不使用 wikilink>

## 待确认

- <不影响核心含义和主要接续的次要疑点>
```
