# 单词卡模板

用于创建或重建普通单词卡。语法、发音对比、音素对比、生活口语句卡不使用本模板。

## 使用规则

- 创建前先搜索重点词汇层，再搜索基础词库层，避免重复建卡。
- `headword` 保持干净词头；`reading` 保持干净假名读音；不要把重音符号写进这两个字段。
- **新字段**：`pronunciation` 存储完整读音（如 `にもつ①`），`variants` 存储汉字差分对（如 `["複/复"]`）。
- **兼容字段**：旧卡的 `reading` + `accent_display` 和 `kanji_diff_pairs` 仍可正常使用，系统会自动回退读取。
- `accent_display` 必须存在。未知时留空；可靠时写假名加重音，如 `에도つ①`、`じてんしゃ②／⓪`。
- `accent_display` 有值时，在正文 `## 核心` 同步写 `- 重音：<accent_display>`；未知时保留 `- 重音：`。
- `confusable_with` 和正文 wikilink 只指向真实存在的词卡。尚未创建但值得补充的易混项写入 `## 待补卡`，使用普通文字。
- 每张词卡默认 2-4 个高频搭配；例句优先由搭配扩展，不写孤立例句。
- 动词重点记录助词框架，如 `Nを預ける`、`Nに向かう`。
- 名词重点记录常搭配动词，如 `荷物を預ける`、`荷物を受け取る`。
- 形容词重点记录常修饰名词或常见句型，如 `濃い味`、`Nが濃い`。
- 常用搭配不可靠时宁可少写，不硬编。核心词义、读音或词头不可靠时，暂停建卡并确认。
- 汉字差分只记录具体干扰字形，如 `複/复`；不要记录整词的简体写法。

## 课堂重点词卡模板

保存位置：`学习系统/词库/重点词汇/<headword>.md`

```md
---
track: class_review
item_type: vocab
status: active
priority: normal
done_today: false
headword:
reading:
accent_display:
pronunciation:
variants: []
meaning_zh:
source_notes:
  - "[[笔记/YYYY.M/YYYY.M.D]]"
first_seen: YYYY-MM-DD
last_seen: YYYY-MM-DD
seen_count: 1
error_count: 0
review_stage: day0
next_review: YYYY-MM-DD
last_reviewed: ""
confusable_with: []
kanji_diff: false
kanji_diff_pairs: []
tags:
  - jp/vocab
  - jp/class_review
---

# <词头>（<reading>）

## 快速复习

- 中文：
- 读音：
- 常用搭配：

## 核心

- 重音：
- 词性 / 语感：
- 使用场景：

## 常用搭配与例句

- 搭配：<常用搭配>
  - 例句：<由搭配扩展的自然日语例句>

## 易错 / 易混

## 来源

- [[笔记/YYYY.M/YYYY.M.D]]
```

```md

headword:

accent_display:

  - "[[笔记/YYYY.M/YYYY.M.D]]"

promote_candidate: false

kanji_diff_pairs: []
tags:
  - jp/vocab
  - jp/base_vocab
  - jp/class_review
---

# <词头>（<reading>）

## 核心

- 重音：
- 中文：
- 词性 / 语感：

## 常用搭配与例句

- 搭配：<常用搭配>
  - 例句：<由搭配扩展的自然日语例句>

```

## 按需区块

有实际内容时，将以下区块放在 `## 来源` 之前；没有实际内容时省略。

```md
## 待补卡

- <尚不存在但值得后续创建的易混词，不使用 wikilink>

## 待确认

- <不影响核心词义、读音和词头的次要疑点>
```
