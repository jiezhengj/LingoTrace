# Template Index

Copy the template below to the appropriate folder, then fill in the fields. `review_stage` uses `day0 / day1 / day3 / day7 / day14 / day30 / day90 / day180 / mastered`.

## Daily Study Checklist Template

Recommended to copy directly into the daily note.

Reference: [[系统配置/模板/en/每日学习清单模板]]

Use cases:

- Record what was actually done today
- Note blockers and breakpoints
- Leave an actionable entry point for tomorrow

## Class Notes Template

## Classroom Vocabulary Template

Full template and rules: [[系统配置/模板/en/单词卡模板]]

- Core content is collocations and natural examples expanded from collocations

## Survival Speaking Card Template

Cards are stored in `学习系统/生活口语/句库/<scene_category>/`, organized by scene subdirectories. Use `<scene>_<core_sentence_or_function>.md` as filename. Long scene materials and video notes go in `学习系统/生活口语/场景指南/`, without `track: survival_speaking`.

Scene categories:

- `home`
- `school_and_study`
- `dining`
- `shopping`
- `errands_and_medical`
- `work`
- `travel`
- `communication_and_invitation`
- `daily_chat`

Frontmatter constraints:

- `scene` uses English snake_case (e.g., `home`, `school`, `small_talk`).
- `speaker_role` only uses `self / staff / other`.
- `function` uses English snake_case, never empty.
- `en_text`, `meaning_zh`, `reply_hint` are required.

```md
---
track: survival_speaking
status: active
priority: normal
done_today: false
scene:
speaker_role: staff
function:
en_text:
meaning_zh:
reply_hint:
audio_ref: ""
source_notes: []
first_seen: YYYY-MM-DD
last_seen: YYYY-MM-DD
seen_count: 1
error_count: 0
review_stage: day0
next_review: YYYY-MM-DD
last_reviewed: ""
tags:
  - en/survival
  - en/speaking
---

## When to Use

Describe the scenario, speaker role, and purpose so the card is immediately clear.

## Say It Directly

Core sentence, replaceable variants, or same-scene alternatives.

## What They Might Say / How I Respond

Choose one heading based on `speaker_role`: cards where you initiate use `## What They Might Say`; staff/other-initiated cards use `## How I Respond`.

## Notes

Omit if there are no register, politeness, or boundary concerns.

## Listen & Repeat Audio

Include embed only when there is a reliable, precisely matching audio slice; omit the entire section if not available.

## If I Didn't Understand

Only include 1-2 rescue phrases most suitable for this scene.
```
