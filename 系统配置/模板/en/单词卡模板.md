# Vocabulary Card Template

Use this template to create or rebuild standard vocabulary cards. Grammar, pronunciation contrast, and phoneme contrast cards do not use this template.

## Usage Rules

- Search the focus vocabulary layer first, then the base vocabulary layer, to avoid duplicate cards.
- `headword` keeps the clean word form; `pronunciation` stores the IPA or other pronunciation notation.
- `variants` stores spelling variants (e.g., `color/colour`, `realise/realize`).
- Each card defaults to 2-4 high-frequency collocations; examples are expanded from collocations, not isolated sentences.
- Verbs prioritize particle/preposition frames (e.g., `depend on`, `look forward to`).
- Nouns prioritize common verb collocations (e.g., `make a decision`, `take a photo`).
- Adjectives prioritize common modified nouns (e.g., `heavy rain`, `strong opinion`).
- If reliable collocations are unavailable, write fewer rather than inventing them.

## Optional Sections

Include these sections before `## Sources` when there is actual content; omit when empty.

