# Grammar Card Template

Use this template to create or rebuild grammar cards under `学习系统/语法/`.

## Usage Rules

- Search for exact-name and comparison candidates before creating a grammar card.
- When a grammar card has a real comparison target, populate `contrast_with`.
- Write `formation` as a YAML list, even for single-formation patterns.
- Provide two natural examples per usage branch by default.
- Keep examples in English; add short Chinese translations only for difficult sentences.
